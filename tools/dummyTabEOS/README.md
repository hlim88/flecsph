### Tabulated ideal EOS for stellar collapse reader

## Prerequsite
python >= 3.0

Usual modules : numpy, scipy, h5py

### Run the script
Simply type
```
python TabGamma.py
```
or simply type
```
TabGamma.py
```
If you make it executable. 

Once you successfully run the script, you will see `sc_eos_gamma_1p4.h5` which
you can use to test readers.
